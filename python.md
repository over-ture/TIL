#### Python 이론

**표현식(Expression)**

- 하나의 '값'으로 평가될 수 있는 모든 코드

**표현식 예시**

- 3 + 5 : 8 (값)
- x > 10 : T/F (값)
- 5 * 4 : 20 (값)

**값(Value)**

- 표현식이 평가된 결과
- 더 이상 계산되거나 평가될 수 없는, 프로그램의 가장 기본적인 데이터 조각

**표현식과 값**

- 모든 값은 그 자체로 가장 단순한 형태의 표현식
- 모든 표현식이 값인 것은 아니다
 - '3 + 5'는 '표현식'이지만, '평가'를 거쳐야 '값'이 된다

**변수(Variable)**

- **값**을 나중에 다시 사용하기 위해, 그 **값**에 붙여주는 고유한 이름

**변수 할당(Variable assignent)**

- 표현식이 만들어 낸 값에 / 이름을 붙이는 과정 (연결)
    - 할당 연산자: 오른쪽 표현식의 평가 결과 값을 왼쪽 변수에 저장

**변수명 규칙**

 - 영문 알파벳, 언더스코어(_), 숫자로 구성
 - 숫자로 시작할 수 없음
 - 대소문자를 구분
 - 파이썬 내부 예약어는 변수명으로 사용할 수 없음

**변수, 값 그리고 메모리**

1. (1/3)
    - 거리에 집 주소가 있듯 메모리의 모든 위치에는 그 위치를 고유하게 식별하는 메모리 주소가 존재

2. (2/3)
    - 고유한 ID
    - 타입 (Type)
    - 값 (Value)
    - 객체 (Object) : 값, 타입, 그리고 관련된 행동까지 하나로 묶인 개념, 객체는 파이썬에서 다루는 모든 데이터의 실체

3. (3/3)
    - 변수는 특정 개체를 '가리키는(refer/point to)' 이름표    

**변수는 "객체를 가리키는 이름" 이다.**

**재할당(Reassigment)**

- 변수는 특정 값을 '기억' 하거나 '가리키는' 이름
- 재할당은 이 변수가 가리키는 대상을 새로운 값으로 변경하는 행위
- 재할당이 이루어지면, 변수는 이전 값을 완전히 잊고 새로운 값만 기억하게 된다.

**타입**
- 변수나 값이 가질 수 있는 데이터의 종류를 의미
    - 2가지 요소로 구분: '값'과 '값에 적용할 수 있는 연산'
    - **Data Types**
        - 값의 종류와 그 값으로 할 수 있는 '동작(연산)'을 결정하는 속성
        - 필요한 이유: 각 타입에 따라 가능한 기능과 연산이 다르기 때문

**숫자형 데이터**

- 프로그래밍에서 값을 계산하고, 수량을 세고, 데이터를 분석하는 등 가장 기본이 되는 데이터 타입
    - 정수형
    - 실수형
    - 지수 표현법
        - 1,230,000,000 = 1.23e9
        - 0.00314 = 3.14e-3

**산술 연산자**
| 기호 | 설명 |
| :---: | :---: |
| + | 덧셈 |
| - | 뺄셈 |
| * | 곱셈 |
| / | 나눗셈 |
| // | 몫 나눗셈 |
| % | 나머지 |
| ** | 거듭제곱 |
| - | 음수 부호 |

**Sequence Types**

- 여러 개의 값들을 순서대로 나열하여 저장하는 자료형
    - 각 데이터는 기차 칸처럼 순서대로 연결
    - 모든 칸에는 0번부터 시작하는 고유한 번호(인덱스)가 붙어 있다
    - 이 번호를 통해 원하는 칸의 데이터에 바로 접근할 수 있다

**시퀀스 타입의 공통 특징**

1. 순서(Order)
    - 값들이 순서대로 저장 (정렬X)

2. 인덱싱(Indexing)
    - 각 값에 고유 번호를 가지고 있으며, 인덱스를 사용하여 특정 위치의 값을 선택하거나 수정할 수 있음

3. 슬라이싱(Slicing)
    - 인덱스 범위를 조절해 전체 데이터 중 원하는 부분만 값을 잘라내서 사용할 수 있음

4. 길이(Length)
    - len()함수를 사용하여 저장된 값의 개수(길이)를 구할 수 있음

5. 반복(Iteration)
    - 반복문을 사용하여 각 값을 하나씩 순서대로 꺼내 사용할 수 있음

**str(문자열)**

- 문자들의 순서가 있는 변경 불가능한 시퀀스 자료형

1. 만들고 표현하는 법
    1. 따옴표 활용 1
        - 작은따옴표(') 또는 큰따옴표(")로 감싸서 표현
        ```python
        # Hello, World!
        print('Hello, World!')

        #str
        print(type('Hello, World!'))
    
    2. 따옴표 활용 2
        - 따옴표 안에 다른 따옴표를 넣고 싶을 때는 서로 다른 종류의 따옴표를 사용
        ```python
        # 문자열 안에 "큰따옴표"를 사용하려면 작은 따옴표로 묶는다.
        print('문자열 안에 "큰따옴표"를 사용하려면 작은 따옴표로 묶는다.')
        ```
    
    3. 특수 표현
        - 이스케이프 시퀀스 (Escape Sequence)
        - 역슬래시(\)와 문자를 조합해 특별한 기능을 수행 (e.g. 줄바꿈, 탭)
        ```python
        # 따옴표 앞에 \를 붙여 문자로 인식시킴
        print('He\'s a boy.')
        ```

**인덱스**
- 시퀀스 자료형에서 각 값의 위치를 식별하기 위해 부여된 고유한 번호
    - 인덱스는 시작점으로부터 떨어진 거리를 의미
    - '첫 번째 인덱스'는 0이다

- 파이썬은 다른 언어와 달리 음수 인덱스를 지원
    - -1은 맨 마지막 값을 의미, -2는 마지막에서 두 번째 값
    - 이 기능은 마지막 요소를 확인할 때 매우 편리합니다

**슬라이싱**

- 대괄호 [] 안에 시작 위치, 끝 위치, 스텝 표기

**시퀀스의 불변성**

- 시퀀스의 요소는 변경될 수 없음

- 변경하려면?
    - 기존 문자열의 일부와 새 값을 조합하여 새로운 문자열을 만들어야 함

**문장(Statement)**

- 특정 '동작(action)'을 지시하는, 실행 가능한 코드의 최소 단위
