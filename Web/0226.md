### Bootstrap 활용

- **Typography**

  - 제목, 본문 텍스트, 목록 등
  ```HTML
  <div class="display-1">Display 1</div>

  <p>You can use the mark tag to
    <mark>highlight</mark> text.
  </p>

  <ul class="list-unstyled">
    <li>Unstyled list item1</li>
    <li>Unstyled list item2</li>
  </ul>
  ```
